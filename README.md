# Data Structures in Java

Implementation of fundamental data structures and algorithms using Java.

## Included Data Structures
- Dynamic Array
- Ordered Array
- Unordered Array
- Singly Linked List
- Sorted Linked List
- Stack
- Queue
- Circular Queue
- Priority Queue
- Binary Search Tree
- Hash Tables

## Algorithms
### Searching
- Linear Search
- Binary Search

### Sorting
- Bubble Sort
- Selection Sort
- Insertion Sort
- Advanced Sorting Algorithms

## Technologies Used
- Java
- NetBeans IDE
